***Технiчний опис завдань***

Це домашнє завдання є комплексним, бо охоплює такі теми:

- **Тема 1.** Лінійна алгебра. Вектори, матриці та операції
- **Тема 2.** Аналітична геометрія. Довжина, відстань та проєкції
- **Тема 3.** Системи рівнянь та застосування

Необхідно буде розв’язати різні 4 задачі:

- **Задача 1.** Робота з зображеннями як з матрицями
- **Задача 2.** Пошук схожих фільмів
- **Задача 3.** Аналіз виробництва
- **Задача 4.** Прогноз тренду

Умови задачі описані поряд із кодом у файлах `ipynb`.
